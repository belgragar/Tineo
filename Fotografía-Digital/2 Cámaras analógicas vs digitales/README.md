# 📷 1. Cámaras analógicas
- Usan carrete.  
- Cada disparo tiene un coste.  
- No permiten ver la foto al momento.  
- Requieren revelado químico.  

# 📸 2. Cámaras digitales
- Usan un **sensor** (CCD o CMOS).  
- Permiten ver la foto al instante.  
- Guardan miles de fotos en una tarjeta.  
- Permiten editar fácilmente.  
- No hay coste por disparo.

# ❓ 3. ¿Qué cámara elegir?
Depende de:
- Presupuesto  
- Uso (paisaje, producto, retrato, redes sociales)  
- Disponibilidad (muchos alumnos usarán móvil)

## 📊 Tipos de cámaras digitales:
- **Compactas:** pequeñas, fáciles de usar.  
- **Bridge:** más zoom, más controles.  
- **Réflex (DSLR):** profesionales, intercambian objetivos.  
- **Sin espejo (mirrorless):** ligeras, muy avanzadas.  
- **Móviles:** cada vez más potentes.
