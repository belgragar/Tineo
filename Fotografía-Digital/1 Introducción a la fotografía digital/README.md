# 📘  **Introducción a la Fotografía Digital**

La fotografía digital es el proceso de capturar imágenes utilizando una **cámara digital o un teléfono móvil**, almacenarlas en formato electrónico y poder editarlas o compartirlas fácilmente.

A diferencia de la fotografía analógica, que utiliza carretes y procesos químicos, la fotografía digital convierte la luz en **información electrónica**.

> [!important]
> **¿Qué es una imagen digital?**
> 
> Una imagen digital está formada por miles o millones de **píxeles**.
> 
> Un píxel es el punto más pequeño que forma una imagen.

---

# 1. 📦 Tamaño del archivo y compresión

El tamaño del archivo de una imagen es un factor crucial que influye en su almacenamiento, manejo y calidad. Varios elementos determinan este tamaño, cada uno desempeñando un papel importante en cómo se percibe y utiliza la imagen:
- Resolución  
- Formato (JPG, PNG, RAW)  
- Nivel de compresión aplicado 

Comprender estos factores es esencial para optimizar el uso de imágenes en diferentes contextos, desde la captura hasta la edición y la publicación.

El **nivel de compresión** se refiere al grado en que se reduce el tamaño de un archivo al eliminar datos innecesarios o redundantes. Este proceso es fundamental en la gestión de imágenes y otros tipos de archivos, ya que permite optimizar el almacenamiento y facilitar la transferencia de datos. A continuación, se explican algunos aspectos clave sobre el nivel de compresión:

## Tipos de Compresión
1. **Compresión con Pérdida**:
   
    - **Descripción**: Elimina algunos datos de la imagen que son menos perceptibles al ojo humano.
    - **Ejemplo**: Formatos como JPEG utilizan compresión con pérdida, lo que reduce significativamente el tamaño del archivo pero puede afectar la calidad de la imagen si se aplica en exceso.
      
2. **Compresión Sin Pérdida**:
    - **Descripción**: Reduce el tamaño del archivo sin perder ninguna información. Todos los datos originales se pueden recuperar.
    - **Ejemplo**: Formatos como PNG y TIFF utilizan compresión sin pérdida, lo que permite mantener la calidad de la imagen, aunque el tamaño del archivo puede ser mayor que en la compresión con pérdida.

## Importancia del Nivel de Compresión
- **Almacenamiento**: Un mayor nivel de compresión reduce el tamaño del archivo, lo que ahorra espacio en discos duros y servidores.
  
- **Velocidad de Carga**: Los archivos más pequeños se cargan más rápido en páginas web y aplicaciones, mejorando la experiencia del usuario.
  
- **Calidad Visual**: Es crucial encontrar un equilibrio entre el tamaño del archivo y la calidad visual de la imagen. Una compresión excesiva puede resultar en imágenes pixeladas o borrosas.

>[!TIP]
>  Al elegir el nivel de compresión, es importante considerar el propósito de la imagen y el contexto en el que se utilizará.
>
>Para impresión o trabajos profesionales, puede ser preferible utilizar formatos sin pérdida, mientras que para la web, la compresión con pérdida puede ser más adecuada para optimizar tiempos de carga.

# 2. 🖼️ Píxeles y resolución
- **Píxel:** unidad mínima de una imagen digital.  
- **Resolución:** cantidad de píxeles que tiene una imagen.  
  - Cuantos más píxeles, más detalle.  
  - Se expresa como *ancho × alto* (por ejemplo: 1920 × 1080).

> Ejemplo práctico: Una foto de 4000 × 3000 píxeles tiene **12 millones de píxeles** → 12 MP.

## Guía Básica de Resoluciones de Imagen

| Categoría                              | Resolución                  | Descripción                                                                                   |
|----------------------------------------|-----------------------------|-----------------------------------------------------------------------------------------------|
| **1. Resoluciones para Pantallas y Video** |                             |                                                                                               |
|                                        | HD (720p)                  | 1280 x 720 píxeles. Estándar mínimo de alta definición, usado en streaming básico.          |
|                                        | Full HD (1080p)            | 1920 x 1080 píxeles. Común en monitores de PC y videos de YouTube. Ofrece gran nitidez.    |
|                                        | 2K (QHD / 1440p)           | 2560 x 1440 píxeles. Popular entre editores de fotos y gamers, más espacio de trabajo.      |
|                                        | 4K (UHD)                   | 3840 x 2160 píxeles. Cuatro veces más píxeles que el Full HD, estándar para fotografía profesional. |
| **2. Resoluciones en Redes Sociales**  |                             |                                                                                               |
|                                        | Cuadrado (1:1)             | 1080 x 1080 píxeles. Típico de posts de Instagram.                                          |
|                                        | Vertical (9:16)            | 1080 x 1920 píxeles. Usado en Stories, Reels y TikTok.                                      |
|                                        | Horizontal (16:9)          | 1920 x 1080 píxeles. Miniaturas de YouTube o cabeceras.                                    |
| **3. Resolución para Impresión**      |                             |                                                                                               |
|                                        | 72 ppp                      | Suficiente para imágenes solo en pantallas.                                                  |
|                                        | 300 ppp                     | Estándar de calidad para impresión. Necesario para evitar imágenes pixeladas.                |
| **4. Megapíxeles en Cámaras**         |                             |                                                                                               |
|                                        | 12 MP                       | Aproximadamente 12 millones de píxeles (ej. 4000 x 3000 px). Más megapíxeles permiten recortes. |

> [!TIP]
> **Web/Móvil:** Full HD (1080p) es suficiente.
> 
> **TV/Cine:** 4K es el estándar de alta calidad.
> 
> **Impresión:** Asegúrate siempre de alcanzar los 300 ppp.

# 3. 📁 Formatos de imagen
- **JPG:** ligero, ideal para redes sociales y web.  
- **PNG:** más calidad, permite transparencia.  
- **RAW:** archivo sin procesar, máxima calidad, ideal para edición.

## Guía de Formatos: De la Captura a la Edición Final

| Formato      | Uso                                        | En Edición                                                                                                   |
|--------------|--------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| **1. RAW / DNG**   | Captura Profesional                       | Punto de partida para edición seria. Contiene toda la información del sensor, formato "no destructivo".      |
| **2. PSD**         | Photoshop Document                       | Estándar de la industria para edición compleja. Permite guardar capas, máscaras y objetos inteligentes.       |
| **3. PNG**         | Diseño gráfico y exportación web       | Valor en el canal alfa (transparencia). Compresión sin pérdida, ideal para recortes y colocaciones.          |
| **4. JPEG**        | Distribución Estándar                   | Formato "cerrado". Calidad se degrada al editar y guardar. No recomendado como formato de trabajo intermedio. |
| **5. WebP**        | El nuevo estándar web                   | Sustituto moderno del JPEG y PNG. Ofrece transparencias y compresión superior, ideal para web.               |
| **6. TIFF**        | Archivo e Impresión                     | Formato final para imprimir en gran formato. Soporta capas y garantiza color y detalle idénticos al original. |
| **7. HEIF / HEIC** | Captura en Smartphones                  | Formato nativo en iPhone y Android. Profundidad de color de 10 o 12 bits, permite degradados más suaves.      |

Para conversión de formatos se puede usar la herramienta [shutterencoder] (https://www.shutterencoder.com/) (hay que descargar e instalarla en el equipo).

> [!TIP]
> 
>  Resumen para el flujo de trabajo:
> - Captura en RAW o HEIC.
> - Edita y guarda tu proceso en PSD o TIFF (para no perder capas).
> - Exporta para compartir en JPEG (uso general), PNG (si hay transparencias) o WebP (para web).
