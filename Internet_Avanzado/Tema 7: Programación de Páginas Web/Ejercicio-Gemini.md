Eres un experto programador de páginas web. Crea el código para una página web en html5. Todo el código ha de estar en un único archivo.html. La página web será para mostrar información sobre un pueblo, en este caso Tineo. Las caraceterísticas son las siguientes:
Tonle un título *“Mi pueblo: Tineo”*  
La página principal debe incluir:
- Un **título grande** con el nombre del pueblo.  
- Un **párrafo introductorio** explicando dónde está y por qué es especial.  
- Una **imagen representativa** (plaza, paisaje, iglesia, etc.).  
- Un **botón** que lleve a otra pestaña del sitio (por ejemplo, “Historia del pueblo”).
Crea una pestaña llamada **“Historia”**.
Debe contener:
- Un texto  explicando algún aspecto histórico del pueblo (antiguo oficio, tradición, origen del nombre, etc.).  
- Una **galería de 3 imágenes** relacionadas con la historia o lugares emblemáticos.  
- Un **enlace externo** a una fuente fiable (ayuntamiento, Wikipedia, blog local, etc.).
Crea una página llamada **“La parroquia”** o **“Lugar destacado”**.

Incluye:

- Una breve descripción del edificio o lugar.  
- Una **imagen** del sitio.  
- Un **mapa insertado** con su ubicación exacta.  
- Un pequeño texto explicando por qué es importante para el pueblo.
