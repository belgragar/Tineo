# 🌐 Conceptos de Creación de Páginas Web

> **Curso de Iniciación a la Informática**
> *Convierte tu idea en una página web, sin necesidad de ser experto.*

---

## 🧩 1. Conceptos Básicos en la Creación de Web

Para crear una página web, necesitas entender tres partes clave:

### 🎨 1. **Diseño**
Es cómo se ve tu página: colores, imágenes, tamaño de letras, etc.
- Se puede hacer **a mano** con códigos (HTML, CSS), pero es difícil.
- O **automáticamente** con herramientas sencillas.

### 🧰 2. **CMS (Content Management System)**
Es un sistema que **te permite crear y gestionar tu página sin escribir código**.
- Ejemplo: WordPress (sí, el mismo que usa muchos blogs).
- Funciona como un editor de textos, pero para páginas web.

### 🛠️ 3. **Creador Web**
Es una herramienta **simple y visual** que permite crear páginas web sin saber nada de programación.
- Arrastra y suelta imágenes, textos, botones.
- Muy útil para crear un sitio web pequeño (un menú de restaurante, un perfil personal, etc.).

> 💡 **Analogía:** Pintar una casa.
> - El **diseño** es cómo quieres que se vea (azul, con una terraza).
> - El **CMS** es el sistema que te da herramientas para añadir paredes, puertas y ventanas.
> - El **creador web** es un juego de bloques que ya vienen preparados y solo tienes que ponerlos en su sitio.

---

## ☁️ 2. Hosting Gratuito: ¿Qué es y qué hay que saber?

El **hosting** es como el "almacén" donde se guardan **todos los archivos de tu página web** (texto, fotos, videos).

### 🆓 ¿Qué es el hosting gratuito?
Es un espacio en internet donde puedes **colocar tu página web sin pagar nada**.
- Puedes probar si te gusta crear páginas web.
- Ideal para proyectos pequeños o aprendizaje.

### ✅ Ventajas
- **Gratis:** No pagas nada.
- **Fácil de usar:** Muchas plataformas lo ofrecen automáticamente.
- **Sin conocimientos técnicos:** No necesitas saber de servidores.

### ❌ Inconvenientes
- **Publicidad:** Aparecen anuncios en tu página.
- **Rango limitado:** No puedes subir muchos archivos o imágenes grandes.
- **Sin dominio propio:** Tu página tendrá una dirección extraña (ej: `miweb.ejemplo.com`).
- **Puede desaparecer:** Si no lo usas mucho, podrían cerrarlo.

> 💡 **Ejemplo:** Google Sites ofrece hosting gratuito con una cuenta de Gmail.

---

## 🔗 3. Dominio Propio: ¿Qué es y qué hay que considerar?

El **dominio** es el nombre de tu página web (por ejemplo, `miempresa.com`).

### ✅ Ventajas
- **Más profesional:** Suena mejor que una dirección con "gratis".
- **Más confianza:** La gente cree más en un sitio con un nombre propio.
- **Control total:** Tú decides el nombre y no dependes de otros.

### ❌ Inconvenientes
- **Cuesta dinero:** Normalmente entre 10€ y 20€ al año.
- **Tiene que renovarse:** Si no pagas, pierdes el nombre.
- **Necesitas un hosting:** Para tener un dominio, también necesitas un "almacén" (hosting).

> 💡 **Consejo:** Si solo quieres probar, empieza con hosting gratuito. Si vas a usarla para negocio o servicios, mejor adquiere un dominio propio.

---

## 🔗 4. ¿Qué es Filezilla?

**Filezilla** es un programa que te permite **subir y bajar archivos** de tu página web (como fotos, textos, videos) desde tu ordenador hacia el "almacén" (el hosting).

### ¿Para qué se usa?
- Para actualizar tu página (ej: subir una nueva foto de tu restaurante).
- Para hacer una copia de seguridad de tus archivos.
- Para instalar programas en tu web (como un sistema de reservas).

> ⚠️ **Advertencia:** Filezilla es un programa técnico. Si no sabes exactamente qué haces, puedes dañar tu página. ¡Usalo con cuidado!

## 🛠️ 5. Ejemplos de Creadores Web

Estas plataformas son como "juegos de construcción" para páginas web.

| Plataforma | ¿Para quién? | Ventaja |
|---|---|---|
| **Wix** 🟠 | Personas que quieren un sitio profesional (negocios, perfiles personales). | Muy visual, muchos diseños listos. |
| **Jimdo** 🟢 | Emprendedores o pequeños negocios. | Muy sencillo, ideal para empezar. |
| **Ionos** 🔵 | Empresas que necesitan un sitio con funciones avanzadas. | Ofrece hosting, dominio y herramientas de marketing. |
| **Squarespace** 🔵 | Creativos que quieren un diseño único (artistas, diseñadores). | Muy bonito, pero más caro. |

> 💡 **Importante:** Todas estas plataformas tienen versiones gratuitas (con publicidad) y de pago (sin publicidad, con dominio propio).

---

## 🌐 6. Google Sites: El Opción Gratuita y Segura

**Google Sites** es una herramienta **gratuita y segura** que ya viene con tu cuenta de Google.

### ✅ Ventajas
- **Gratis:** No pagas nada.
- **Fácil de usar:** Funciona como Google Docs, pero para páginas.
- **Seguro:** No hay publicidad ni riesgo de virus.
- **Integrable:** Puedes añadir videos de YouTube, formularios, fotos.

### ❌ Inconvenientes
- **Muy básico:** Solo para páginas sencillas (un menú, una presentación).
- **No personalizable:** Pocas opciones de diseño.
- **Sin dominio propio:** Tu página será `sites.google.com/...`.

> 💡 **Recomendación:** Si quieres una página web sencilla y gratuita, **Google Sites** es tu mejor opción.
